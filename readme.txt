Easily add drawing smooth signature field to Contact Form 7 form

== Description ==
Contact Form 7 signature (CF7 signature) plugin helps you to add signature fields.

### Add a signature field to Contact Form 7 ###
Add `sign` tag from wp-admin > contacts > slect form > add tag `[sign]`
Example: `[sign your-sign]`

**Note: Never wrap with `<label>` tag.**

**Good:-**
**`<label>` Signature `</label>`**
[sign* sign-568]

**bad:-**
**`<label>` Signature 
[sign* sign-568]`</label>`**

Then add input field name `[your-sign]` in contact form 7 > mail > attachments section.

== Installation ==

This plugin requires the Contact Form 7 plugin.

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin feature
2. Activate the plugin
3. Edit a form in Contact Form 7
4. Choose "Sign field" from the Generate Tag dropdown
5. Follow the instructions on the page
